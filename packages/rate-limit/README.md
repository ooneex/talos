# @talosjs/rate-limit

API rate limiting middleware with configurable throttling strategies, sliding window counters, and per-client request quota enforcement

## Installation

```bash
bun add @talosjs/rate-limit
```

## Documentation

Read the full documentation at [docs.talosjs.com/components/rate-limit](https://docs.talosjs.com/components/rate-limit).
