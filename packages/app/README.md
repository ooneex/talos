# @talosjs/app

Full-featured application framework for Bun — orchestrates routing, middleware pipelines, dependency injection, caching, logging, and WebSocket support in a modular architecture

## Installation

```bash
bun add @talosjs/app
```

## Documentation

Read the full documentation at [docs.talosjs.com/getting-started/create-app](https://docs.talosjs.com/getting-started/create-app).
